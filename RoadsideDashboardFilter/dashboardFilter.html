Customer Connect AU Dashboard Filter

About this feature

The AU Dashboard Filter is a Chrome extension script that automatically filters the Customer Connect dashboard to only show dispatches in Australia (AU). It enhances usability by removing non-AU rows and sorting AU dispatches by ETA for quick prioritization.


Key features include:

  • Automatically removes non-AU dispatch rows from the dashboard.
  • Displays a live countdown timer with a progress bar indicating the next refresh.
  • Shows the Current Active Dispatches count inline for AU jobs, updated every second.
  • Sorts AU jobs by ETA, prioritizing negative, zero, and positive ETA values accordingly.

The script activates automatically on page load and interacts directly with the dashboard.
