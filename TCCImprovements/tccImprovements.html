TCC QOL Improvements

RS Dashboard Country Filter

On the roadside dashboard, automatically filters to Australia (AU) or New Zealand (NZ), hides non-matching rows, and sorts remaining dispatches by ETA.


Key features:

  • Country dropdown (Australia / New Zealand) next to Stage
  • Types the country code into the dashboard Search bar only
  • Hides non-matching rows (does not delete them)
  • Live countdown timer and active dispatch count
  • Sorts jobs by ETA (highest urgency first)

Activates only on the roadside dashboard when TCC QOL Improvements is enabled. The old Teams dropdown filter remains disabled.
