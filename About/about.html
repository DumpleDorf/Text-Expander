About This Tool

Purpose

This extension serves as a custom built way to help increase efficiency of day-to-day workflows by adding new tools to exising resouces.

Features

  • Text expander tool
  • AU/NZ quick tire quoting tool
  • SCA auto messager tool
  • Custom-built notifier for Towbook

Version

The current version is: 2.0.3

Support

For additional support, please reach out to Darian George on Teams, or Ronan Preston for assistance with the SCA Auto Messager tool.


If you would like to help contribute to the development of this tool, please also reach out to Darian George and we can share the source code.
