SCA AutoMessager

About this feature

SCS AutoMessager allows service to send messages in the Communication Log that are specific to each customer and service center site by pulling data such as name, scheduled date and location.


The tool embeds a button natively within SCA which, when pressed, automatically enters the formatted message for the service team member.


This tool currently only supports the first initial contact message sent to customers, but with support, will be able to support multiple types of messages, such as:

  • First contact and check-in instructions
  • In-Service update messages
  • Checkout/pick-up instructions
